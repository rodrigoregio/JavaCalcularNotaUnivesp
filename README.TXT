Calcule quantos pontos você precisa para passar na prova da Univesp. Ele
pede as notas das atividades semanais e calcula quantos pontos você
precisa para passar na matéria. 
------------------------------------------------------------------------

TITULO DO PROJETO: Calculadora de Nota
OBJETIVO DO PROJETO: Auxiliar o aluno da Univesp a saber quantos pontos
precisa tirar (no mínimo) para passar na matéria.
VERSÃO ou DATA: 1.0.0
COMO INICIAR O PROJETO: Clique no arquivo CalculaNotasUnivesp.jar, se não
executar clique no arquivo executar.bat
